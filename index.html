Manual Técnico: Blog Interactivo con Animación Canvas

Estructura Integrada y Ejercicios Prácticos de Nivel Básico | Feria "El Baratillo"

1. Arquitectura y Archivos del Sistema

Este manual integra HTML5, CSS3, JavaScript y la API de Canvas para construir un entorno interactivo dinámico. El alumno deberá crear un directorio organizado con la siguiente estructura de archivos:

Archivo requerido Tecnología Propósito del Módulo
index.html HTML5 Estructura y esqueleto del blog, definición de categorías de la feria y lienzo Canvas.
estilos.css CSS3 Estilos de diseño, diseño adaptativo básico (Grid/Flexbox) y efectos visuales reactivos (Hover).
script.js JS / Canvas Lógica de interacción con productos y bucle de animación para el movimiento del logotipo.

4. Lógica de Interacción y Animación Canvas script.js

Este módulo realiza dos tareas fundamentales: controla la actualización dinámica de textos en el DOM (cuando se pulsa un producto) e implementa un Bucle de Animación (Loop) usando un control de rebote básico para mover el texto en el Canvas.

// --- VARIABLES DE ANIMACIÓN (Nivel Básico) ---
let posX = 200;       // Posición horizontal inicial (centro del lienzo)
let velocidad = 1;    // Dirección y cantidad de píxeles a mover por cuadro (1 = derecha, -1 = izquierda)

// --- BUCLE DE ANIMACIÓN DE CANVAS ---
function animarLogo() {
    const canvas = document.getElementById('logoCanvas');
    const ctx = canvas.getContext('2d');

    // 1. Limpiar completamente el lienzo en cada fotograma
    ctx.clearRect(0, 0, canvas.width, canvas.height);

    // 2. Dibujar el fondo del letrero (Fijo)
    ctx.fillStyle = '#ef4444'; 
    ctx.fillRect(10, 10, 380, 60);

    // 3. Dibujar el marco brillante interno (Fijo)
    ctx.strokeStyle = '#f59e0b'; 
    ctx.lineWidth = 4;
    ctx.strokeRect(15, 15, 370, 40);

    // 4. Actualizar la variable de posición horizontal
    posX = posX + velocidad;

    // 5. CONTROL DE REBOTE (Pirueta Izquierda -> Derecha -> Izquierda)
    // Si la posición supera un límite derecho o izquierdo, multiplicamos por -1 para invertir el rumbo
    if (posX > 270 || posX < 130) {
        velocidad = velocidad * -1; 
    }

    // 6. Dibujar el texto en la nueva coordenada 'posX' calculada
    ctx.fillStyle = '#ffffff';
    ctx.font = 'bold 22px Arial';
    ctx.textAlign = 'center';
    ctx.fillText('FERIA "EL BARATILLO"', posX, 45);

    // 7. Solicitar al navegador que renderice el siguiente cuadro de animación de forma fluida
    requestAnimationFrame(animarLogo);
}

// Iniciar el bucle gráfico al cargar la ventana por completo
window.onload = animarLogo;


// --- INTERACTIVIDAD DINÁMICA DE LA INTERFAZ ---
function mostrarDetalle(nombre, precio, descripcion) {
    const contenedorInfo = document.getElementById('vista-previa');
    
    // Inyección de plantillas HTML para actualizar el panel de información
    contenedorInfo.innerHTML = `
        
${nombre}

        
Precio Especial: ${precio}

        
${descripcion}

        
    `;
}
💡 Concepto Clave para el Alumno: La función ctx.clearRect() es el borrador físico del lienzo. Si se comenta esa línea, el texto se dibujará sobre sí mismo infinitamente, generando un rastro sólido.

5. Cuaderno de Ejercicios Prácticos (Nivel Básico)

Proponga a los alumnos las siguientes modificaciones sencillas sobre el código base para evaluar su comprensión de la lógica multimedia e interactiva:

🛠️ Ejercicio 1: Modificación de la Velocidad de la Pirueta

Objetivo: Identificar qué variables controlan el tiempo y ritmo en un entorno multimedia.

Instrucción: Ve a tu archivo script.js, localiza la variable velocidad y cámbiala del valor 1 actual al valor 3. Guarda y prueba en el navegador.

  • ¿El texto va más rápido o más lento?
  • ¿Qué ocurre si cambias el valor a un número decimal muy pequeño como 0.5?

🎨 Ejercicio 2: Personalización del Color Multimedia

Objetivo: Relacionar diseño en Canvas con propiedades estéticas de CSS.

Instrucción: Cambia los colores del logo de la feria para que se adapte a una temática nocturna:

  1. En el archivo script.js, cambia el color de fondo del rectángulo (ctx.fillStyle = '#ef4444';) por un tono azul oscuro o morado (ejemplo: #2e1065).
  2. En el mismo archivo, cambia el color del borde decorativo a verde brillante (#22c55e).

📦 Ejercicio 3: Expansión del Catálogo de Feria

Objetivo: Desarrollar habilidades de copia, edición y comprensión de paso de parámetros funcionales en HTML.

Instrucción: Añade un producto nuevo a cualquiera de las dos categorías en tu archivo index.html.

  • Crea un botón extra debajo del producto "Encendedor".
  • Usa la misma estructura y asegúrate de cambiar los parámetros de la función mostrarDetalle() para que muestre un nombre, precio y descripción diferentes (por ejemplo: una "Sartén antiadherente").
